# 🎮 Game Bắn Gà (Chicken Shooter)

Đây là một game 2D đơn giản được phát triển bằng Unity, trong đó người chơi điều khiển tàu vũ trụ để tiêu diệt các con gà địch và ghi điểm.

---

## 🕹️ Cách chơi

Bạn điều khiển nhân vật (tàu vũ trụ) để di chuyển và bắn hạ kẻ địch.

---

## 🎮 Điều khiển

| Phím / Chuột | Chức năng |
|--------------|----------|
| W | Di chuyển lên |
| S | Di chuyển xuống |
| A | Di chuyển sang trái |
| D | Di chuyển sang phải |
| Chuột trái | Bắn đạn |

---

## ✨ Tính năng chính

- Điều khiển nhân vật bằng phím WASD
- Hệ thống bắn đạn bằng chuột trái
- Enemy (gà) di chuyển và xuất hiện liên tục
- Va chạm giữa đạn và kẻ địch
- Hệ thống tính điểm cơ bản
- Gameplay đơn giản, dễ hiểu

---

## 🛠️ Công nghệ sử dụng

- Unity (2D)
- C#
- Physics2D của Unity
- Prefab system
